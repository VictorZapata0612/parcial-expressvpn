# Parcial Maquetación Web

Proyecto de maquetación realizado como parte del parcial de desarrollo web.

## Descripción

Se desarrolló la maquetación de una landing page basada en el diseño proporcionado por el profesor.  
El proyecto utiliza **HTML5** y **CSS3** para construir la estructura y estilos de la página.

## Tecnologías utilizadas

- HTML5
- CSS3
- Git
- GitHub

## Estructura del proyecto

```
parcial-vpn
│
├── index.html
├── css
│ └── styles.css
│
├── assets
│ └── images
│
└── README.md

```


## Instalación / Uso

1. Clonar el repositorio:

git clone <[text](https://github.com/VictorZapata0612/parcial-expressvpn)>


2. Abrir el archivo `index.html` en el navegador.

## Flujo de trabajo con Git

El proyecto incluye múltiples commits y ramas para simular un flujo de trabajo real de desarrollo.

- `feature/maquetacion`
- `fix/css-spacing`
- `docs/readme`

Cada rama incluye cambios específicos que posteriormente se integran mediante **Pull Request**.
